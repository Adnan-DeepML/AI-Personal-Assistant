# 🤖 AI Telegram Assistant

An AI-powered Telegram assistant built with **n8n** and **Google Gemini** that automates everyday tasks through a conversational interface.

## Features

- 💬 Text & voice message support
- 🎤 Voice transcription
- 🧠 Conversation memory
- 📧 Read & send Gmail emails
- 📅 Create Google Calendar events
- 📊 Access contacts from Google Sheets
- 📚 Wikipedia search
- ⚡ AI-powered automation

## Workflow

Telegram → AI Agent → Gemini → Connected Tools → Response

The assistant automatically detects text or voice messages, processes the request, calls the required tools, and sends the result back to Telegram.

## Tech Stack

- n8n
- Google Gemini
- Telegram Bot API
- Gmail API
- Google Calendar API
- Google Sheets API
- Wikipedia API
